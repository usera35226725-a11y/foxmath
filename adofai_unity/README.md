### Game Details & Guide
Adofai: A Comprehensive Guide



Game Overview


Adofai is a browser-based puzzle game that challenges players to rotate tiles to match pairs of adjacent tiles.



How to Play


To play, simply navigate through the game and click on each tile to rotate it. The objective is to create pairs of matching tiles.



    

Controls:


    

        

Clicking on a tile:
 Rotates the tile.

        

Moving left and right:
 Swaps two adjacent tiles in the same direction.

        

Moving up and down:
 Swaps two adjacent tiles in opposite directions.
